Study Syllabus

A session-by-session plan for facilitating the Ex4CSLearning CTF study. Each session combines a mini-lesson, guided challenges, and reflection activities.

Standards aligned: All lessons map to the Alabama K–12 Computer Science Standards (2021) and the NICE Cybersecurity Workforce Framework (2020). See the Standards Mapping page for full details.
// session 1
Session 1 · OSINT

OSINT & Ethics

Introduction to cybersecurity and the ethical use of publicly available information.

Learning Activities

  • Review the Code of Conduct and discuss responsible online research.
  • Students complete the Pre-Test to establish baseline knowledge.
  • Mini-lesson: "What is a CTF?" — overview of challenges and real-world applications.
  • Complete the OSINT Challenge Set on CTFd.
Find the Bean World Traveler Vanished Motto

Assessment

  • Pre-Test completion (Qualtrics)
  • Submission of OSINT challenge flags in CTFd
  • Brief class discussion on ethical OSINT use

Resources

  • OSINT Learning Lab
  • CTFd Portal
// session 2
Session 2 · Cryptography

Cryptography & Secret Messages

Understanding encryption, encoding, and how to decode hidden messages using classical ciphers.

Learning Activities

  • Mini-lesson: substitution vs. transposition ciphers with hands-on examples.
  • Students complete the Cryptography Challenge Set on CTFd.
  • Group discussion: "How does encryption protect data in everyday life?"
Secret Scroll Hidden Codebook Headmaster's Secret

Assessment

  • Challenge flag submissions in CTFd
  • Participation in group reflection

Resources

  • Cryptography Learning Lab
  • CTFd Portal
// session 3
Session 3 · Passwords

Password Security & Reflection

Password hashing, salting, cracking ethics — and wrapping up with the post-test and group reflection.

Learning Activities

  • Mini-lesson: password entropy and hashing — how passwords are stored securely.
  • Complete the Password Security Challenge Set on CTFd.
  • Class reflection and completion of the Post-Test.
  • Group discussion: "What makes a password secure, and why does ethical hacking matter?"
How Long Until It's Gone County Salt Coffee Shop Vibes

Assessment

  • Challenge flag submissions in CTFd
  • Post-Test completion (Qualtrics)
  • Teacher & student feedback form submission

Resources

  • Password Security Learning Lab
  • Feedback Forms
