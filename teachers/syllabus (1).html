Module Syllabus

A full outline of the Cyber Security Learning Module — three sequential modules combining mini-lessons, CTF challenges, and guided reflection, all mapped to Alabama K–12 CS standards.

// module overview

The Three Modules

Module 01

OSINT & Ethics

Focus: Ethical use of information & digital investigation // learning activities
  • Review the Code of Conduct and discuss responsible use of online resources
  • Students complete the Pre-Test to gauge baseline Cyber Security understanding
  • Mini-lesson: What is a CTF? — overview of challenges and real-world applications
  • Tutorial videos: Intro to OSINT, search tools, and verification ethics
  • Complete the OSINT Challenge Set: Find the Bean, World's Traveler, Vanished Motto
  • Debrief discussion on digital ethics and responsible online investigation
Assessment:

Pre-Test completion + OSINT challenge submissions

Pre-Test →
Module 02

Cryptography

Focus: Understanding encryption and decoding techniques // learning activities
  • Mini-lesson: substitution vs. transposition ciphers with hands-on examples
  • Tutorial videos: Intro to Cryptography, hashing vs. encryption, Base64, ciphers
  • Complete the Cryptography Challenge Set: Secret Scroll, Hidden Codebook, Headmaster's Secret
  • Group discussion: How does encryption protect data in everyday life?
Assessment:

Evidence of challenge completion + group reflection participation

Challenges →
Module 03

Password Security & Reflection

Focus: Password hashing, strength, and cracking ethics // learning activities
  • Mini-lesson: password entropy and hashing — how passwords are stored securely
  • Tutorial videos: hashing, how hackers crack passwords, rainbow tables, strong passwords
  • Complete Password Security Challenge Set: How Long Until It's Gone, County Salt, Forgotten Coffee Shop Vibes
  • Class reflection and completion of the Post-Test
  • Group discussion: What makes a password secure and why does ethical hacking matter?
Assessment:

Post-Test completion + reflection form submission
