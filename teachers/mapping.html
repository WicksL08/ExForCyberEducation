OSINT Challenges

Challenge Points / Level Alabama DLCS NICE Framework Learning Focus
Find the Bean! 200 — Medium R1R2R3R5R6 ANCO Combine OSINT tools (images, maps, websites) to track down a target object; evaluate credibility of sources.
The World's Traveler 100 — Easy R1R2R5R6 ANCO Use search tools and public sources to answer location-based clues; practice safe, ethical online research habits.
The Vanished Motto 300 — Hard R1R2R3R5R6 ANCOIN Follow a multi-step OSINT trail (Wayback, WHOIS, news); document evidence and reasoning like a cyber investigator.
// module 02

Cryptography Challenges

Challenge Points / Level Alabama DLCS NICE Framework Learning Focus
The Secret Scroll 100 — Easy R3R4R6 SPPR Recognize why encryption is used to protect information; practice decoding a simple cipher using step-by-step reasoning.
The Hidden Codebook 200 — Medium R3R4R6 SPPRAN Compare different substitution ciphers; analyze patterns in ciphertext; connect cryptography to real-world secure communication.
The Headmaster's Secret 300 — Hard R3R4R5R6 SPPRAN Work collaboratively to break a multi-step cipher; justify solution process; relate strong/weak encryption to security risk.
// module 03

Password Security Challenges

Challenge Points / Level Alabama DLCS NICE Framework Learning Focus
How Long Until It's Gone 50 — Easy R1R2R3R6 PRAN Explore how password length and complexity affect cracking time; connect to personal password hygiene.
County Salt 100 — Easy R1R2R3R6 PRAN Understand hashing and salting; see why salted hashes are harder to crack; relate to stored passwords in real systems.
Forgotten Coffee Shop Vibes 100 — Easy R1R2R3R4R6 PRAN Analyze a realistic password reuse scenario; reason about weak vs. strong passwords and user behavior.
