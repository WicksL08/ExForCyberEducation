| Term | Definition | Example |
|------|-------------|----------|
| **Encoding** | Convert data to a readable format | Base64 |
| **Hashing** | One-way transformation (fixed-length) | SHA-256 |
| **Encryption** | Two-way protection using keys | AES, RSA |
| **Salting** | Add randomness to hashes | Password + salt |
| **Rainbow Table** | Pre-computed hashes used to crack weak passwords | Hash lookup attack |
| **OSINT** | Gathering public data ethically | Searching open sources |
