// key vocabulary

Terms to Know

Term Definition Example
EncodingConvert data to a readable format — not a security measureBase64
HashingOne-way transformation producing a fixed-length outputSHA-256
EncryptionTwo-way protection using keys — data can be decryptedAES, RSA
SaltingAdding randomness to a password before hashingpassword + random_salt → hash
Rainbow TablePre-computed hashes used to crack unsalted passwordsHash lookup attack
OSINTGathering public data ethically from open sourcesSearch engines, maps, archives
Digital FootprintTraces left behind when using the internetTagged photos, public posts
MetadataHidden data in files — dates, locations, device infoGPS coords in a photo
