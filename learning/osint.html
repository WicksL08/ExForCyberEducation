// what is osint

What is OSINT?

Open-Source Intelligence (OSINT) is the practice of collecting and analyzing information from publicly available sources — websites, social media, public records, images, and more. Cyber Security analysts use OSINT to investigate threats, verify identities, and understand digital environments without accessing any private or unauthorized systems.

  • OSINT uses only publicly available, legally accessible information
  • It is used by law enforcement, journalists, researchers, and Cyber Security professionals
  • Ethical OSINT never involves unauthorized access, hacking, or privacy violations
  • Digital footprints — the traces we leave online — are the foundation of OSINT investigation

// key vocabulary

Key Terms to Know

OSINT

Gathering information from publicly available sources without unauthorized access.

Ex: searching social media, public records, maps
Digital Footprint

The trail of data left behind when using the internet — public posts, accounts, locations.

Ex: tagged photos, check-ins, usernames
Verification

Confirming that found information is accurate by cross-referencing multiple sources.

Ex: checking three sources before concluding
Metadata

Hidden data embedded in files that can reveal creation date, location, and device info.

Ex: GPS coordinates in photo files
Reverse Image Search

Finding the original source of a photo by searching using the image itself rather than keywords.

Ex: TinEye, Google Images
Web Archive

A stored snapshot of a website from a previous date — useful for finding deleted content.

Ex: Wayback Machine
