##  What Is OSINT?

**Open-Source Intelligence (OSINT)** means gathering and analyzing information from **public sources** like websites, social media, government records, and online databases.

It’s one of the first skills cybersecurity professionals learn because it teaches how to:
- Investigate **digital footprints**
- Validate **sources and claims**
- Protect **personal and organizational data**

> *Think like a detective, act like a professional.*

---

##  OSINT Process Overview

| Step | Description | Example |
|------|-------------|----------|
| **1. Define the Goal** | What are you looking for? | Find a company’s cybersecurity contact. |
| **2. Gather Information** | Use search tools and public data. | Google, WHOIS, LinkedIn, Twitter. |
| **3. Verify the Source** | Confirm accuracy with multiple sources. | Cross-check images, URLs, timestamps. |
| **4. Analyze & Report** | Summarize findings clearly. | Present verified info with screenshots. |

##  Tools & Techniques

**Search Operators (Google):**
- `site:` → Search a specific website.  
  `site:alabama.edu cybersecurity`
- `filetype:` → Search by file format.  
  `filetype:pdf cyber policy`
- `intitle:` → Search by page title keywords.  
  `intitle:"contact us" site:org`

**Verification Tools:**
- [Google Reverse Image Search](https://images.google.com/){:target="_blank"}  
- [TinEye](https://tineye.com/){:target="_blank"}  
- [Whois Lookup](https://who.is/){:target="_blank"}  
- [URLScan.io](https://urlscan.io/){:target="_blank"}  
- [Wayback Machine](https://archive.org/web/){:target="_blank"}

##  OSINT Ethics & Legal Boundaries

**Allowed:**
- Public websites, open databases, news sources  
- Data intentionally shared publicly  

**Not Allowed:**
- Hacking, guessing passwords, bypassing privacy settings  
- Accessing private or leaked data  

> Always ask: “Would I be okay if someone used this method on me?”

## Key Vocabulary

| Term | Definition |
|------|-------------|
| **OSINT** | Gathering and analyzing public information. |
| **Verification** | Confirming truth using multiple sources. |
| **Digital Footprint** | All traces of online activity. |
| **Metadata** | Hidden data about files (timestamps, locations). |
| **Attribution** | Identifying who created or posted something. |
