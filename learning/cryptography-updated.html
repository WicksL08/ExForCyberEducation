// what is cryptography

What is Cryptography?

Cryptography is the science of protecting information by transforming it into an unreadable format — so only authorized people can decode it. From ancient Caesar ciphers to modern AES encryption, cryptography has been used for thousands of years to protect secrets, secure communications, and keep data safe.

  • Encoding converts data to a different format — it is not a security measure on its own
  • Encryption uses a key to protect data — only someone with the key can decrypt it
  • Hashing is a one-way transformation — it cannot be reversed
  • Modern Cyber Security relies on cryptography to protect passwords, messages, and transactions

// key vocabulary

Key Terms to Know

Encoding

Converting data into a different format — not a security measure, easily reversed.

Ex: Base64 encoding
Hashing

A one-way transformation that produces a fixed-length output. Cannot be reversed.

Ex: SHA-256 hash
Encryption

Two-way protection using keys — data can be encrypted and later decrypted.

Ex: AES, RSA encryption
Caesar Cipher

A substitution cipher that shifts each letter by a fixed number of positions in the alphabet.

Ex: A→D, B→E with shift of 3
Ciphertext

The encrypted or encoded version of a message that cannot be read without decoding.

Ex: "Khoor" = "Hello" in Caesar +3
Plaintext

The original readable message before it has been encrypted or encoded.

Ex: "Hello World"
