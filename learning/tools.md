# Tools for Students 

These tools will help you solve the CTF challenges!  
All of them are **free and work in a web browser**. 
Use them **only for class activities** and follow the [Code of Conduct]({{ '/about/student-agreement/' | relative_url }}).

---

##  OSINT (Open-Source Intelligence)

**These tools help you find and check information online.**

- **Wayback Machine** — Shows old or deleted versions of websites.  
  https://web.archive.org

- **Google Image Search** — Helps find where an image comes from or appears online.  
  https://images.google.com

- **TinEye** — Reverse image search to find the original source of a photo.  
  https://tineye.com

- **Metadata2Go** — Shows hidden details inside files or images, such as date/time and device info.  
  https://www.metadata2go.com/

- **URLVoid** — Checks if a website is safe or suspicious.  
  https://www.urlvoid.com/

- **GetLinkInfo** — Reveals what’s behind a shortened link (like bit.ly).  
  https://www.getlinkinfo.com/

- **IP Location Lookup** — Shows basic information about an IP address.  
  https://www.iplocation.net/ip-lookup

>  *Use OSINT tools only for class clues — never for real people or accounts.*

## Cryptography & Encoding

**These tools help you decode or create secret messages.**

- **CyberChef** — A powerful all-in-one tool for decoding, encoding, hashing, and more.  
  https://gchq.github.io/CyberChef/

- **Cryptii** — Great for testing simple ciphers like Caesar, Bacon, and Morse code.  
  https://cryptii.com/

- **dCode** — A huge library of online cipher solvers for puzzles and codes.  
  https://www.dcode.fr/tools-list

- **Base64 Decode/Encode** — Turns Base64 text into readable words and back.  
  https://www.base64decode.org/

- **ASCII Table** — Shows how letters and symbols are stored as numbers in computers.  
  https://www.asciitable.com/

- **MD5 & SHA Generator** — Creates hash values to show how passwords are stored.  
  https://md5hashing.net/hash

-  **Caesar Cipher Tool** — Shifts letters to decode or encode messages.  
  https://www.boxentriq.com/code-breaking/caesar-cipher

- **Vigenère Cipher Tool** — Decodes messages that use repeating key patterns.  
  https://www.boxentriq.com/code-breaking/vigenere-cipher

> *If text looks scrambled or coded, one of these tools will help crack it.*

## Passwords & Security

**These tools help you understand how passwords work and how to make them stronger.**

- **Password Strength Checker (zxcvbn)** — Shows how strong a password is and how long it would take to guess.  
  https://lowe.github.io/tryzxcvbn/

- **CyberChef** — Also helps you explore hashing and encryption concepts.

- **Password Concepts Page** — Learn about hashing, salting, and secure password habits.  
  {{ '/learning/passwords/' | relative_url }}

> *Never enter real passwords. Create fake sample passwords to test.*

## Networking, Search & Web Investigation  
**(This is where SHODAN & WHOIS go — fully added!)**

**These tools help you explore how the internet works, safely and ethically.**

- **WHOIS Lookup** — Shows who owns a website, when it was created, and basic domain info.  
  Great for checking if a website looks trustworthy.  
  https://who.is/

- **Shodan Search Engine** — A search engine that shows **what devices are online**, like servers, cameras, and routers.  
  You can use it to learn **why exposed devices need security**, but you **cannot** access or control anything.  
  https://www.shodan.io  
  >  *Only search general topics. Never interact with or log into devices.*

- **HTTPBin** — Shows what happens when a website sends or receives information.  
  https://httpbin.org/

- **DNS Checker** — Shows which servers a website uses and how it connects online.  
  https://dnschecker.org/all-dns-records-of-domain.php

- **Explain Shell** — Breaks down command-line commands into plain English.  
  https://explainshell.com/

- **What Is a URL?** — Helps you understand how web addresses work.  
  https://developer.mozilla.org/en-US/docs/Learn/Common_questions/Web_mechanics/What_is_a_URL

## File & Data Helpers

**These tools help you analyze files, text, and URLs.**

- **VirusTotal** — Checks if a file or link is safe.  
  https://www.virustotal.com/

- **FileFormat.com** — Tells you what type of file something is and how to open it.  
  https://www.fileformat.com/

- **Text/Binary Converters** — Converts between binary, text, and numbers.  
  https://www.unitconverters.net/text-tools/

- **Regex101** — Helps test and understand search patterns.  
  https://regex101.com/
